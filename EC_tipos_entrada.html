TIPOS DE ENTRADA

Entrada tipo Texto Input.



Caja para ingresar texto.

Entrada Tipo Correo



Input de correo
valida que la información
dentro del input tenga un formato
de correo.

Entrada de Tipo Numero Telefonico.



El input de teléfono
valida que únicamente
puedas ingresar 10 dígitos,
por medio del maxlength="10".

Entrada de tipo URL



Valida que la URL
tenga ciertas características,
identifica que sea una URL válida.

Entrada de tipo Contraseña



Valida el texto
que se ingrese
como una contraseña.

Entrada de Tipo Seleccion Multiple






En este tipo de input
se pueden crear varias casillas
tipo checkbox.

Entrada de Tipo Selección Unica






Este tipo de input
sirve para crear varias opciones
con solo una posible
seleccion por parte del usuario.

Entrada de Tipo Select




Este tipo de input
sirve para generar por defecto
opciones a elegir para el usuario.

Entrada de Tipo Select con Input





En este tipo de input
se puede tener la opcion
de escribir o seleccionar
alguna de las sugerencias.

Entrada de Tipo Archivo.




Este tipo de input
se utiliza para
seleccionar y subir archivos.

Entrada de tipo Boton.






Este tipo de input
sirve para agregar botones.
